# Patient Flow & Hospital Load Prediction

## Overview
This project is an AI-powered hospital operations analytics system designed to forecast patient arrivals, bed occupancy, and departmental load across ER, OPD, ICU, and Wards.

## Features
- Synthetic hospital data generation
- Patient arrival forecasting
- Bed occupancy prediction
- Department-wise load analysis
- Simulation engine (planned)
- Optimization module (planned)
- Streamlit dashboard

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Statsmodels
- SimPy
- Streamlit
